jQuery - 尺寸

  • width(像素值),height(像素值) 设置或返回元素的宽、高度
  • innerWidth(),innerHeight() 设置或返回元素的宽、高度，包括内边距
  • outerWidth(true),outerHeight(true) 设置或返回元素的宽、高度。无true，包括内边距和边框；有true，包括内边距、边框、外边距

jQuery - 遍历

  • parent() 返回被选元素的直接父元素
  • parents() 返回被选元素的所有上级元素
  • parents("指定元素") 返回被选元素所有上级元素中的指定元素
  • parentsUntil("指定元素") 返回被选元素和指定元素间的的所有上级元素
  • children() 返回被选元素的所有直接子元素
  • children("指定元素") 返回被选元素的所有直接子元素中的指定元素
  • find("*") 返回被选元素的所有后代
  • find("指定元素") 返回被选元素的所有后代中的指定元素
  • siblings("指定元素") 返回被选元素的所有同级元素或其中的指定元素
  • next(),prev() 返回被选元素的下一个同级元素
  • nextAll(),prevAll() 返回被选元素的所有后继同级元素
  • nextUntil("指定元素"),prevUntil("指定元素") 返回被选元素和指定元素之间的所有后继同级元素

jQuery - AJAX get() 和 post()

  • $.get(url,callback) 从指定资源请求数据，在url中数据可见
    1. GET请求可被缓存
    2. GET请求保留在浏览器历史记录中
    3. GET请求可被收藏为书签
    4. GET请求不应在处理敏感数据时使用
    5. GET请求有长度限制
    6. GET请求只应当用于取回数据
  • $.post(url,data,callback) 从指定资源提交要被处理的数据，在url中数据不可见，在HTTP消息主体中
    1. POST请求不会被缓存
    2. POST请求不会保留在浏览器历史记录中
    3. POST请求不能被收藏为书签
    4. POST请求长度没有限制
    5. POST请求可用于发送并取回数据
